# Zero_relationship LWE Secret Sampler Prototype

This repository contains a prototype implementation of the proposed `Zero_relationship` idea:

- baseline: LWE secret sampled from a small discrete Gaussian
- candidate: secret sampled from a nonlinear space generated by random Maclaurin-based functions

## What this prototype does

1. Generates a baseline secret vector with small Gaussian coefficients.
2. Generates a `Zero_relationship` secret vector by:
   - creating `f` random Maclaurin functions (`sin`, `cos`, `exp`, `atan`, `sinh` families)
   - sampling random points on each function
   - connecting points to build a piecewise nonlinear space
   - sampling coefficients from that nonlinear piecewise geometry
3. Resets all random functions and all intervals every time a new secret vector is sampled.
4. Compares baseline vs candidate with several metrics:
   - mean, std, entropy
   - lag-1 correlation and average pairwise dimension correlation
   - zero/non-zero separability AUC under noisy observation

## Quick start

Run from this folder:

```powershell
python .\zero_relationship_lwe.py --samples 500 --n 256 --show-lwe-preview
```

Example with stronger nonlinear diversity:

```powershell
python .\zero_relationship_lwe.py --zr-functions 10 --zr-points 12 --zr-order-min 4 --zr-order-max 10
```

Example focused on lower linear correlation:

```powershell
python .\zero_relationship_lwe.py --samples 300 --n 256 --zr-refresh-prob 0.40
```

Example with multi-seed average comparison (recommended):

```powershell
python .\zero_relationship_lwe.py --benchmark-seeds 16 --samples 120 --n 256
```

Example with stronger zero/non-zero blurring profile:

```powershell
python .\zero_relationship_lwe.py --benchmark-seeds 16 --samples 120 --n 256 --zr-profile blur_heavy
```

## Main parameters

- `--seed`: base random seed
- `--benchmark-seeds`: number of consecutive seeds averaged in one report
- `--n`: secret dimension
- `--m`: number of LWE equations for sample preview mode
- `--q`: modulus for sample preview mode
- `--samples`: number of secret vectors per seed used for metric evaluation
- `--obs-noise-sigma`: noise level for zero/non-zero separability check
- `--baseline-sigma`: sigma for baseline small Gaussian sampler
- `--error-sigma`: sigma for LWE error in preview mode
- `--secret-bound`: integer coefficient clipping bound
- `--zr-functions`: number of random Maclaurin functions per secret generation
- `--zr-points`: random points sampled per function
- `--zr-order-min`, `--zr-order-max`: truncation order range of Maclaurin series
- `--zr-profile`: predefined profile (`default`, `balanced`, `blur_heavy`)
- `--zr-refresh-prob`: probability of rebuilding local nonlinear space while sampling one secret vector
- `--zr-zero-to-nonzero`: probability of switching exact zero to +/-1 near boundary mixing
- `--zr-unit-to-zero`: probability of switching +/-1 to zero near boundary mixing
- `--zr-boundary-mix`: extra randomization over `{-1, 0, 1}` for low-magnitude coefficients

## Security note

This is a research prototype, not a security proof and not production cryptography.

Changing the secret distribution may affect both security assumptions and solver behavior in non-trivial ways. Use formal cryptanalysis and large-scale attack benchmarking before making security claims.
